## SEO Checklist for New Packages

- [ ] Package ID contains primary keyword
- [ ] Description includes 3-5 target keywords naturally
- [ ] All 20 tags used with relevant keywords
- [ ] README has keyword-rich headings
- [ ] README includes code examples (improves engagement)
- [ ] Icon is professional and recognizable
- [ ] Source Link enabled (symbol package)
- [ ] GitHub topics match NuGet tags
- [ ] Repository description matches package focus
